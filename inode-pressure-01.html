Filesystem Full by Inodes

This lesson is about filesystems that are full by file count, not by byte usage.

Situation

A spool path is blocked by too many tiny obsolete files.

Mission Goal

  • Prove inode pressure
  • Find the obsolete tiny-file path
  • Clean it without touching live files

What You Should Learn

Low disk usage in bytes does not rule out filesystem exhaustion. Inode pressure is a different failure mode.
